# TimeTracking System (Система учета рабочего времени)

Веб-приложение для фиксации трудозатрат сотрудников, разработанное в рамках производственной практики в **ООО «НОРБИТ»**.

## 🚀 Функциональные возможности
- **Учет времени:** Списание часов на конкретные задачи с привязкой к проектам.
- **Интерактивный UI:** Динамический стикер с цветовой индикацией дневной нагрузки (8 часов).
- **Отчетность:** Просмотр истории проводок за день или за календарный месяц.
- **Управление структурой:** Создание проектов и задач через административный интерфейс.
- **Валидация:** Серверная проверка лимита времени (не более 24 часов в сутки).

## 🛠 Технологический стек
- **Backend:** .NET 8, ASP.NET Core Web API.
- **Database:** Entity Framework Core (MS SQL Server / SQLite).
- **Frontend:** Vue.js 3 (Composition API), Axios, Bootstrap 5.
- **API Documentation:** Swagger UI.

## 📦 Структура проекта
- `/Controllers` — REST API эндпоинты для проектов, задач и записей времени.
- `/Models` — Сущности базы данных и DTO.
- `/Data` — Контекст базы данных и миграции.
- `/wwwroot` — Клиентская часть (HTML, CSS, JS).
